# Site Let's Code

Projeto desenvolvido no curso de Full Stack Python da EBAC - Escola Britânica de Artes Criativas e Tecnologia.

Site de informações sobre linguagens de programação com aplicação de Bootstrap avançado com ferramentas como Dropdown, Navbar, Carousel de imagens, Modal e Cards.
Bootstrap aplicado no projeto com os arquivos minificados JS e CSS utilizados localmente.

## Incluir Bootstrap via CDN

Adicionar o links abaixo no seu arquivo HTML.

### Arquivo minificado CSS dentro da tag <head>

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">

### Arquivo minificado JS abaixo de todo projeto antes do fechamento da tag <body>

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>

* Sem validação de informações e imagens e também formulário de contato. Apenas dados superficiais para o projeto.

## 🚀 Começando

Estas instruções permitirão que você obtenha uma cópia do projeto em execução em sua máquina local para fins de desenvolvimento e teste.

- Ao obter o projeto em sua máquina local, certifique-se de manter todos os arquivos no mesmo local.
- Abra o arquivo index.html em seu browser (navegador) para visualizar a página.

## 🛠️Ferramentas utilizadas para construção do projeto

* **HTML5** - Linguagem de marcação utilizada na construção de páginas na Web.
* **CSS3** - Cascading Style Sheets é um mecanismo para adicionar estilos a uma página web.
* **Bootstrap** - Framework web para desenvolvimento de componentes de interface e front-end usando HTML, CSS e JavaScript.
